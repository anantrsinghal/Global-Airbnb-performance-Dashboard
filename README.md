
Global Airbnb Performance Dashboard

Overview
This is an end-to-end Power BI dashboard project built using the Airbnb dataset from Maven Analytics. The dashboard analyzes listing performance, customer reviews, pricing, host distribution, and seasonal trends across major cities.

The project focuses on applying Power BI concepts to solve business problems through interactive visualizations and meaningful insights.

 Features

- KPI Dashboard
- Market Share Analysis
- Pareto Analysis
- Pricing Comparison
- Ratings Heatmap
- Review Analysis
- Seasonality Analysis
- Interactive Dashboard Navigation

 Tools Used

- Power BI
- DAX
- Power Query
- Data Modeling

 Dataset

Source: Maven Analytics

The dataset contains Airbnb listings, hosts, reviews, property types, prices, and review ratings from multiple cities.

 Key Learnings

- Built an interactive Power BI dashboard
- Created DAX measures and KPIs
- Applied Pareto analysis to identify key markets
- Analyzed pricing, customer ratings, and seasonal trends
- Practiced data storytelling through business-focused visualizations
